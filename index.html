O que é?

Rails é um framework baseado na linguagem Ruby, e é o mais difundido/utilizado por sua comunidade, chegando a ser confundido com a própria linguagem.

Arquitetura

O framework é baseado na arquitetura MVC (Model - View - Controller) que já é bastante difundida e utilizada por inúmeros frameworks já presentes no mercado.

Model

Camada geralmente utilizada para provir e gerenciar as regras de negócio da aplicação.

Ex: Camada responsável por retornar dados de uma tabela banco de dados

View

Camada de apresentação responsável por mostrar os dados presentes do Model referenciado.

Ex: Camada responsável por mostrar a listagem dos dados providos pelo Model ao usuário

Controller

Camada responsável por fazer a comunicação do Model com a View, importante lembrar que nessa camada não devemos ter nada referente a camada de negócio, deixando assim com a sua camada responsável (Model).
